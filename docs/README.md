# Documentación del Script

Este documento explica cómo funciona el script `math_operations.py` y cómo usarlo.

## Descripción

El script `math_operations.py` es una calculadora básica que realiza operaciones matemáticas simples, como suma y resta. Está diseñado para demostrar el flujo de trabajo colaborativo en Git y GitHub.
